# PHPFixer Website

Static marketing site built with Phanstatic + Vite.

## Setup

```bash
composer install
npm install
```

## Build

```bash
npm run build:all
```

This builds the PHP static site and compiles assets (TailwindCSS + Alpine.js).

## Stack

- **Phanstatic** - PHP static site generator
- **Vite** - Asset bundler
- **TailwindCSS v4** - Styling
- **Alpine.js** - Interactivity

See `CLAUDE.md` for detailed development documentation.
